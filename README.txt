For coders
- cd to this folder
- Run `yarn`
- Run `yarn start:user` to track changes in user folder only
- Run `yarn start:user` to track changes in admin folder only
- If you wish to prettify your HTML and lint your css before commit, run `npm test`
- `yarn test` will run on pre-commit so don't forget to got commit --amend esp. when you have new changes in pug file. 

Included in this boilerplate
- Atomic Design template
- Stop Scrolling of body when opening modal esp in IOS devices
